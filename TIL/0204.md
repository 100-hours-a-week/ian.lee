오늘 numpy 와 pands 에 관하여 배움

| 단어 | 정의  |
| --- | --- |
| Numpy | 파이썬 라이브러리로 대용량 수치형 데이터의 연산을 효율적으로 하게 도와준다.  |
| 차원 | 데이터 축(axis)의 개수로, 자료구조가 몇 개의 벡터로 이뤄져있는지 나타낸다.  |
| 형태 | 각 축에 있는 요소의 개수를 튜플 형태로 표현한 것  |
| 인덱스 | 특정 value를 정수형, 문자형 등의 값으로 위치를 나타내는 것 |
| 유니버설 함수 | 효율성(시간 복잡성, 공간 복잡성)을 고려하여 배열의 크기가 서로 달라도 연산을 해주는 함수 |
| Pandas | 파이썬의 라이브러리로 데이터를 분석하거나 수정 변환 하는 용도로 사용한다. |
| 시리즈 | 데이터 프레임의 하위 자료구조 형태로 한개의 열만 있는 자료 |
| 데이터프레임 | 정형 데이터와 같이 Key, Value를 기준으로  구성된 자료구조 |
| Grouping | 데이터를 통계 연산을 통해 데이터를 묶어 분석에 용이하게 함 |
| Merging | 특정 열(한개 이상)을 기준으로 두개 이상의 데이터 프레임을 하나로 합치는 함수 |
| 피벗 | 데이터 형태를 축을 바꿔 보기 쉽고 사용 용도에 맞게 만듬 |
